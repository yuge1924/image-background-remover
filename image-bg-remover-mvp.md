# 📄 MVP 需求文档
**产品名称：** Image Background Remover
**版本：** v1.0 MVP
**日期：** 2026-03-24

---

## 一、产品定位

一个轻量、免安装的在线图片背景去除工具。用户上传图片，一键获得透明背景的 PNG 文件，无需注册，无需下载软件。

**目标用户：**
- 电商卖家（产品图处理）
- 设计师（快速抠图）
- 普通用户（证件照、头像处理）

---

## 二、技术架构

| 层级 | 技术选型 |
|------|---------|
| 前端 | HTML + Tailwind CSS（或 Next.js） |
| 后端 | Cloudflare Workers |
| AI API | remove.bg API |
| 存储 | 无（内存处理，不落盘） |
| 部署 | Cloudflare Pages + Workers |

---

## 三、核心功能（MVP 范围）

### 3.1 图片上传
- 支持拖拽上传 / 点击选择文件
- 支持格式：JPG、PNG、WEBP
- 文件大小限制：≤ 12MB（remove.bg 限制）
- 上传后实时预览原图

### 3.2 背景去除
- 点击「Remove Background」按钮触发处理
- 调用 remove.bg API 处理
- 处理中显示 Loading 状态
- 失败时展示友好错误提示

### 3.3 结果展示与下载
- 处理完成后展示对比图（原图 / 去背景图）
- 去背景图以棋盘格背景展示透明区域
- 提供「Download PNG」按钮，直接下载透明 PNG

### 3.4 重置
- 提供「Try Another Image」按钮，清空状态重新开始

---

## 四、页面结构（单页）

```
[ Header ] — Logo + 产品名 + 简短说明

[ 上传区域 ]
  - 拖拽区 / 点击上传
  - 支持格式说明

[ 处理区域 ]（上传后显示）
  - 原图预览
  - → Remove Background 按钮
  - Loading 动画

[ 结果区域 ]（处理完后显示）
  - 原图 | 去背景图 对比
  - Download PNG 按钮
  - Try Another Image 按钮

[ Footer ] — 简单版权信息
```

---

## 五、非功能需求

| 项目 | 要求 |
|------|------|
| 响应时间 | 去背景处理 ≤ 10s（受 remove.bg 影响） |
| 移动端 | 响应式布局，支持手机浏览器 |
| 安全性 | API Key 存于 CF Worker Secrets，不暴露前端 |
| 隐私 | 图片仅在内存中转，不存储，不记录 |
| 浏览器兼容 | Chrome / Safari / Firefox 最新版 |

---

## 六、MVP 不包含（后续迭代）

- ❌ 用户注册 / 登录
- ❌ 批量处理
- ❌ 自定义背景色 / 背景图替换
- ❌ API 对外开放
- ❌ 使用量统计 / 付费墙
- ❌ 图片编辑功能

---

## 七、成功标准

- [ ] 用户可以上传图片并成功获得透明背景 PNG
- [ ] 全流程无需注册，< 3 步完成操作
- [ ] 移动端可正常使用
- [ ] 页面加载时间 < 2s
- [ ] remove.bg API Key 不泄露
